## Dataset Information

This project uses the CICIDS-2017 dataset for training and evaluation of
IoT intrusion detection models.

### Dataset Source
CICIDS-2017 dataset is provided by the Canadian Institute for Cybersecurity (CIC).

Official download link:
https://www.unb.ca/cic/datasets/ids-2017.html

### Notes
- The dataset is not included in this repository due to its large size
  (~1.5 GB) and licensing constraints.
- Users should download and merge the CSV files as per CIC guidelines.
